# Vk avatar

## Overview
Script for downloading user's avatar from [Vk](https://vk.com).

## How to use 
To execute help:
```
python2 main.py --help
```
```
usage: main.py [-h] [-u URL]

optional arguments:
  -h, --help         show this help message and exit
  -u URL, --url URL  Link to VK account
```
